# پورتفولیو حرفه‌ای - توسعه‌دهنده وب

یک وب‌سایت پورتفولیو استاتیک حرفه‌ای با تم فضایی و کهکشانی که مهارت‌های توسعه وب را به نمایش می‌گذارد.

## 🌟 ویژگی‌ها

- **طراحی فضایی**: تم کهکشانی با انیمیشن‌های ستاره‌ها و سیارات
- **چندزبانه**: پشتیبانی از زبان‌های ترکی، انگلیسی و فارسی
- **ریسپانسیو**: سازگار با تمام دستگاه‌ها (موبایل، تبلت، دسکتاپ)
- **انیمیشن‌های پیشرفته**: انیمیشن‌های نرم و جذاب
- **سازگار با SEO**: بهینه‌سازی شده برای موتورهای جستجو
- **سرعت بالا**: بارگذاری سریع و بهینه
- **رابط کاربری مدرن**: طراحی زیبا و حرفه‌ای
- **Django تخصصی**: مهارت‌های پیشرفته در Django و Python

## 🚀 تکنولوژی‌های استفاده شده

- **HTML5**: ساختار معنایی و مدرن
- **CSS3**: استایل‌دهی پیشرفته با Flexbox و Grid
- **JavaScript**: انیمیشن‌ها و تعاملات
- **Bootstrap 5**: فریم‌ورک CSS برای ریسپانسیو بودن
- **Django**: فریم‌ورک وب قدرتمند Python
- **Python**: زبان برنامه‌نویسی اصلی
- **Font Awesome**: آیکون‌های زیبا
- **Google Fonts**: فونت‌های چندزبانه (Inter, Vazirmatn)

## 📁 ساختار پروژه

```
static-portfolio/
├── index.html          # صفحه اصلی

├── js/
│   └── main.js         # اسکریپت‌های JavaScript
└── README.md           # مستندات پروژه
```

## 🛠️ نصب و راه‌اندازی

### روش 1: اجرای مستقیم
1. فایل‌ها را دانلود کنید
2. فایل `index.html` را در مرورگر باز کنید

### روش 2: استفاده از سرور محلی
```bash
# با Python
python -m http.server 8000

# با Node.js
npx serve .

# با PHP
php -S localhost:8000
```

### روش 3: GitHub Pages
1. پروژه را در GitHub آپلود کنید
2. در Settings > Pages، Source را روی `main` تنظیم کنید
3. وب‌سایت در `https://username.github.io/repository-name` قابل دسترسی خواهد بود

## 🌍 پشتیبانی چندزبانه

این وب‌سایت از سه زبان پشتیبانی می‌کند:
- **ترکی** (پیش‌فرض): زبان اصلی وب‌سایت
- **انگلیسی**: برای مخاطبان بین‌المللی
- **فارسی**: برای مخاطبان فارسی‌زبان

### تغییر زبان
- از دکمه‌های زبان در گوشه راست بالای صفحه استفاده کنید
- ترجیح زبان در localStorage ذخیره می‌شود
- تمام محتوا، فرم‌ها و متا تگ‌ها به‌روزرسانی می‌شوند

## 🎨 شخصی‌سازی

### تغییر اطلاعات شخصی
فایل `index.html` را ویرایش کنید:
- نام و عنوان
- توضیحات
- اطلاعات تماس
- لینک‌های شبکه‌های اجتماعی

### اضافه کردن زبان جدید
در فایل `js/main.js`:
1. ترجمه‌های جدید را به آبجکت `translations` اضافه کنید
2. دکمه زبان جدید را در HTML اضافه کنید
3. فونت مناسب را در CSS تنظیم کنید

### تغییر رنگ‌ها
در فایل `css/style.css`:
```css
:root {
    --primary-color: #74b9ff;
    --secondary-color: #a29bfe;
    --accent-color: #fd79a8;
}
```

### اضافه کردن پروژه‌ها
در بخش Projects در `index.html`:
```html
<div class="col-lg-4 col-md-6 mb-4">
    <div class="project-card">
        <div class="project-image">
            <img src="path/to/image.jpg" alt="نام پروژه">
        </div>
        <div class="project-content">
            <h3>نام پروژه</h3>
            <p>توضیحات پروژه</p>
            <div class="project-tech">
                <span class="tech-tag">تکنولوژی</span>
            </div>
        </div>
    </div>
</div>
```
